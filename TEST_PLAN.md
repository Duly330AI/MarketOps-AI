# MarketOps AI - Real Data MVP Test Plan

## Goal
Evaluate whether the app is useful as a daily AI-assisted market research and paper-trading tool. Do not evaluate real-money profitability yet.

## Test Start Date
2026-06-29

## Test Watchlist
* Stocks: NVDA, MSFT, AMD, AAPL
* ETF: SPY
* Crypto: BTC-USD, ETH-USD

## Daily Routine
* Open dashboard.
* Check `dataQuality` for all assets.
* Run AI analysis only when needed, not repeatedly.
* Record observations.
* Do not overtrade.

## Forecast Routine
* Create 7-day and 30-day forecasts for selected assets.
* Forecasts must be evaluated only by the app’s resolved forecast engine.
* Do not manually edit resolved results.

## Paper Trading Routine
* Start with virtual cash only.
* Create only a few test trades.
* Record reason for every trade.
* No real money.
